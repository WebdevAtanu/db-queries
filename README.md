This repository contains comprehensive study notes and reference materials for various Database Management Systems (DBMS). It is designed as a practical learning resource for developers, database administrators, and anyone looking to strengthen their understanding of SQL and NoSQL databases.

## Database Systems Covered

### Relational Databases

- **MSSQL (Microsoft SQL Server)** - Covers T-SQL syntax, stored procedures, triggers, indexing, and advanced query techniques.

- **PostgreSQL** - Includes PostgreSQL-specific SQL commands, constraints, and functions.

- **MySQL** - Focuses on MySQL syntax, data types, and query optimization techniques.

### NoSQL Databases

- **MongoDB** - Covers JavaScript-based queries, aggregation pipelines, and document-based operations.

## Contents Overview

### MSSQL Notes

- **Basic Operations**: Database and table creation, CRUD operations
- **Advanced Topics**: Stored procedures, triggers, cursors, dynamic SQL
- **Functions**: String, date, and aggregate functions
- **Query Techniques**: Joins, subqueries, CTEs, window functions
- **Administration**: Backup, security, indexing, transactions

### PostgreSQL Notes

- **Core SQL**: Basic queries, table operations, constraints
- **Advanced Features**: Grouping, clauses, and aggregate functions
- **Data Types**: String handling and data manipulation

### MySQL Notes

- **Fundamentals**: Database creation, table operations, data manipulation
- **Data Integrity**: Constraints, unique keys, and default values

### MongoDB Notes

- **Query Operations**: Basic document queries and CRUD operations
- **Aggregation Framework**: Advanced data processing pipelines
- **Data Relationships**: Lookup operations and joins

## Prerequisites

- Access to respective database systems for hands-on practice
- Any text editor or IDE (e.g., VS Code) for exploring SQL/JS files

## 🎯 Purpose

This repository is intended to:

- Serve as a quick reference for database concepts
- Help in interview preparation
- Provide hands-on query examples for practice
